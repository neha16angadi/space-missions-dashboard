Space Missions Dashboard 

A python dashboard to explore space mission data from 1957 onwards. 


Features
  - Filter missions by company, mission status, and date range.
  - View total number of missions and success rate based on selected filters. 
  - Compare missions by company, trends over time, and missions by location.
  - Full mission dataset, ability to filter and sort.

Tech Stack Used
- python : main language used
- dash : ineractive web dashbaord
- plotly express : clear graphs/visuals
- pandas : data filtering
